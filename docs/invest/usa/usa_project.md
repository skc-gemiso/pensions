# 미국 경제지표 — 프로젝트 개요

## 개요

FRED(미국 연방준비제도 경제 데이터), 미국 재무부 TIC, Frankfurter API 환율 데이터를 수집하여
미국 거시경제 주요 지표 추세를 표와 차트로 시각화하는 투자 분석 화면.
수집기는 `pensions/collector/usa/`에 구현 완료.

- 경로: `/invest/usa`
- 상태: **구현 완료**
- 수집기 문서: [usa_collector_task.md](usa_collector_task.md)
- 기술 사양: [usa_task.md](usa_task.md)

---

## 메뉴 구조

```
/invest/usa                         미국 경제지표
├── /invest/usa                     미국 경제 지표 수집 (수집 이력 테이블)
├── /invest/usa/indicator           지표별 최신값 카드 + 시계열 차트
├── /invest/usa/treasury            미국 국채 보유 현황 (일본·중국)
└── /invest/usa/fx                  원/달러 환율 조회
```

## 화면별 기능

### 미국 경제 지표 수집 (`/invest/usa`)

| 기능 | 설명 |
|------|------|
| 수집 안내 카드 | 수집 대상·주기·방법 안내 (전체 사용자) |
| 수집 실행 버튼 | admin 전용 — 수동 수집 트리거 |
| 수집 이력 테이블 | usa_collect_log 최근 60건 — 수집기·대상·시작/완료 시각·상태·건수·메모 + 새로고침 버튼 |

### 지표별 시계열 (`/invest/usa/indicator`)

| 기능 | 설명 |
|------|------|
| 지표 카드 | 7개 FRED 지표 최신값·전월 대비·한국어 설명 카드 그리드 (클릭 시 해당 지표 차트 선택 + 스크롤 이동) |
| 지표 선택 | 드롭다운 또는 카드 클릭 |
| 기간 선택 | 1년 / 2년 / 5년 / 전체 |
| 최신 자료 링크 | Investing.com 해당 지표 페이지 바로가기 버튼 (기간 선택 우측) |
| 시계열 차트 | 바 차트 (Recharts) |
| 상세 테이블 | 날짜, 값, 전기 대비 변화 |

### 미국 국채 보유 현황 (`/invest/usa/treasury`)

| 기능 | 설명 |
|------|------|
| 일본·중국 보유액 비교 | 이중 라인 차트 (USD 기본, KRW 토글 가능) |
| 요약 카드 | USD 금액 강조(primary), KRW 고정환율 환산 참조(secondary) |
| 원화 환산 | 고정 환율 $1 = 1,400원 기준 (착시 방지 — 환율 변동 미반영) |
| 단위 토글 | USD(기본) / KRW 전환 버튼 |
| 기간 선택 | 1년 / 2년 / 5년 / 전체 |
| 상세 테이블 | 날짜, 일본/중국별 금액(달러)·증감(달러)·증감률(달러)·금액(원)·증감(원)·증감률(원) |
| 데이터 안내 | TIC 발표 시차 안내, 원화 참고용 고정환율 경고 표시 |

### 원/달러 환율 조회 (`/invest/usa/fx`)

| 기능 | 설명 |
|------|------|
| 일별 환율 라인 차트 | Recharts + 기간 평균 기준선 |
| 기간 선택 | 1년 / 2년 / 5년 / 전체 |
| 요약 카드 | 최근 환율, 전일 대비, 기간 평균 |
| 상세 테이블 | 날짜, 환율, 전일 대비 변화 |
| 수집 관리 패널 | admin 전용 — 수동 수집 트리거 버튼 |
