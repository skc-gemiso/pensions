# 연금투자 시뮬레이션 — 프로젝트 개요

## 개요

KODEX200 ETF와 KODEX200 타겟위클리커버드콜 ETF에 장기 적립 투자 시,
퇴직 시점의 예상 평가금액·수익률·월 배당금을 연평균 수익률 시나리오별로 비교하는 웹 시뮬레이터.
IRP 의무 비율(안전자산 30% + ETF 70%)을 적용한 IRP 탭도 제공한다.

- 경로: `/sim`
- 상태: **완전 구현**
- 참고 파일: [app/sim/page.tsx](../../app/sim/page.tsx), [app/sim/actions.ts](../../app/sim/actions.ts)
- 기술 사양: [sim_task.md](sim_task.md)

## 핵심 개념

### 두 가지 시뮬레이션 모드

| 탭 | 설명 | 대상 계좌 |
|----|------|-----------|
| 수익율 확인 | 납입액 100% ETF 투자 | 연금저축펀드, ISA |
| IRP 수익율 확인 | 안전자산 30% + ETF 70% 의무 비율 | IRP |

### 시나리오 비교

6개 연평균 수익률(-20%, -10%, 0%, +5%, +10%, +20%)에 대해
KODEX200 ETF와 커버드콜 ETF의 적립 완료 / 퇴직 시점 / 월 배당을 나란히 비교.

### 탭 접근 권한

| 탭 | 접근 권한 |
|----|-----------|
| 수익율 확인, IRP 수익율 확인 | 전체 사용자 |
| 개인 탭 (동민·고은·샤인 등) | `admin`, `khj` 역할만 |
