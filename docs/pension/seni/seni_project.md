# 노령연금 — 프로젝트 개요

## 개요

국민연금 노령연금(일반·조기·연기)의 수급 조건과 감액/증액 기준을 안내하고,
향후 개인 데이터 기반 예상 수령액 계산을 제공하는 화면.

- 경로: `/pension/seni`
- 상태: **진행 중** (안내 UI 구현, 계산 기능 미구현)
- 참고 파일: [app/pension/seni/page.tsx](../../app/pension/seni/page.tsx)

---

## 화면 구조

```
/pension/seni
├── 노령연금 기본 정보
│   ├── 수급 개시 연령 (출생연도별)
│   └── 최소 가입 기간 (10년)
├── 조기노령연금 안내
│   ├── 최대 5년 앞당겨 수령
│   └── 1년당 6% 감액 (최대 30%)
├── 연기노령연금 안내
│   ├── 최대 5년 연기 수령
│   └── 1년당 7.2% 증액 (최대 36%)
└── 예상 수령액 추정 (미구현)
    └── "데이터를 연동하거나 직접 입력하세요" 메시지
```

## 핵심 개념

### 수급 개시 연령 (2024년 기준)

| 출생연도 | 수급 개시 연령 |
|----------|----------------|
| 1952년 이전 | 60세 |
| 1953~1956년 | 61세 |
| 1957~1960년 | 62세 |
| 1961~1964년 | 63세 |
| 1965~1968년 | 64세 |
| 1969년 이후 | 65세 |

### 수령 전략 비교

| 전략 | 기준 | 변화 |
|------|------|------|
| 조기 수령 | 5년 앞당김 | -30% (연 -6%) |
| 정상 수령 | 수급 개시 연령 | 기준 |
| 연기 수령 | 5년 연기 | +36% (연 +7.2%) |

## To-Be 개선 방향

- 국민연금 납부 이력(`/pension/nat`) 연동 → 예상 수령액 자동 계산

- 조기/정상/연기 수령 시나리오별 총 수령액 비교
- 기대 수명 입력 시 최적 수령 시점 제안
