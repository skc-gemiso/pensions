# 국민연금 — 프로젝트 개요

## 개요

국민연금 납부 현황과 예상 수령액 이력을 관리하는 화면.
납부 진행률을 시각화하고, 조회 시점별 예상 수령액(세전/세후)을 기록·비교한다.

- 경로: `/pension/nat`
- 상태: **완전 구현**
- 참고 파일: [app/pension/nat/page.tsx](../../app/pension/nat/page.tsx), [app/pension/nat/actions.ts](../../app/pension/nat/actions.ts)

---

## 화면 구조

```
/pension/nat
├── 납부 진행 현황
│   ├── 가입 시작일 ~ 예상 종료일 진행 바
│   ├── 경과 개월 / 총 예상 개월 표시
│   └── 진행률 (%) 계산
├── 예상 수령액 기록
│   ├── 입력 폼 (확인 시점 / 총 납부액 / 월 수령액 세전·세후)
│   ├── 추가 버튼
│   └── 이력 테이블 (날짜 역순 정렬)
└── (연결) 나의 연금 현황 대시보드 ← /pension/my
```

---

## 핵심 개념

### 납부 진행 현황

- 가입 시작일과 정년 기준 예상 종료일로 총 납부 기간(개월) 산출
- 현재 날짜 기준 경과 개월 계산 → 진행률(%) 표시
- 진행 바(progress bar) 시각화

### 예상 수령액 스냅샷

- 국민연금 홈페이지 등에서 조회한 예상 수령액을 수동으로 기록
- 조회 시점(확인 시점), 총 납부액, 월 수령액 세전/세후 4개 항목
- DB에 저장하여 시점별 변화 추이 확인 가능
