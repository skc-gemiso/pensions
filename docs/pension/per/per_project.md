# 개인연금 — 프로젝트 개요

## 개요

연금저축펀드·IRP·ISA 계좌의 평가액을 한 곳에서 확인하고,
관련 시뮬레이션 도구로 연결하는 허브 화면.

- 경로: `/pension/per`
- 상태: **진행 중** (UI 골조 구현, 데이터 연동 미구현)
- 참고 파일: [app/pension/per/page.tsx](../../app/pension/per/page.tsx)

---

## 화면 구조

```
/pension/per
├── 계좌 현황 카드 (미구현)
│   ├── 연금저축펀드 평가액 (현재: "- 원")
│   ├── IRP 평가액 (현재: "- 원")
│   └── ISA 평가액 (현재: "- 원")
└── 빠른 이동
    ├── 연금저축펀드 시뮬레이션 → /sim
    └── 복리의 마법 → /magic
```

---

## 핵심 개념

### 계좌 유형별 세제 혜택

| 계좌 | 납입 한도 | 세제 혜택 |
|------|-----------|-----------|
| 연금저축펀드 | 연 600만원 | 세액공제 13.2%~16.5% (총급여 5,500만원 기준) |
| IRP | 연금저축 포함 연 900만원 | 세액공제 동일 + 퇴직금 이전 |
| ISA | 연 2,000만원 (5년 누적 1억) | 200만원(서민 400만원) 비과세 / 초과분 9.9% 분리과세 |

### 투자 전략 연결

- 연금저축펀드: `/sim` (KODEX200 ETF 시뮬레이션)
- IRP: `/sim` IRP 탭 (안전자산 30% + ETF 70%)
- 복리의 마법: `/magic`
