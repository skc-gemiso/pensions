# 복리의 마법 — 프로젝트 개요

## 개요

초기 투자금과 월 납입액을 설정하고 연 수익률·투자 기간을 입력하면,
연도별 자산 성장 추이를 차트로 시각화하는 복리 계산기.
복잡한 시나리오 비교 없이 "얼마를 넣으면 얼마가 되는가"를 직관적으로 보여준다.

- 경로: `/magic`
- 상태: **완전 구현**
- 참고 파일: [app/magic/page.tsx](../../app/magic/page.tsx)

---

## 화면 구조

```
/magic (복리의 마법)
├── 시뮬레이션 설정 (왼쪽 패널)
│   ├── 초기 투자금 (원)
│   ├── 월 납입액 (원)
│   ├── 연 수익률 (%)
│   └── 투자 기간 (년)
├── 결과 요약 카드 (왼쪽 하단)
│   ├── 최종 평가액
│   ├── 총 납입액
│   └── 수익 (복리 효과)
└── 자산 성장 추이 차트 (오른쪽)
    ├── 평가액 라인 (파란색)
    └── 납입액 라인 (회색 점선)
```

---

## 핵심 개념

### 복리 계산 방식

월 복리(월말 납입) 가정:
- 매달 잔액에 월 수익률 적용 후 월 납입액 추가
- 연 수익률 → 월 수익률 변환: `r_month = annualRate / 100 / 12`

### 결과 해석

| 항목 | 의미 |
|------|------|
| 최종 평가액 | 투자 기간 종료 시점의 총 자산 |
| 총 납입액 | 초기 투자금 + 월 납입 합계 |
| 수익(복리 효과) | 최종 평가액 − 총 납입액 |
