# 생활비 관리 (`/life/cost`)

## 개요

월별 생활비·고정지출·카드결제를 항목별로 입력·관리하는 개인 가계부 화면.
`/life` 접근 시 `/life/cost`로 자동 리다이렉트.

---

## 화면 구조

### 레이아웃 (2-column)

```
┌─────────────────────┐  ┌──────────────────────────────────────────────────┐
│  월 선택 드롭다운    │  │  고정지출                           [+ 항목추가] │
│  수입 대비 지출 현황 │  │  고정이체 & 금융                    [+ 항목추가] │
│  주요 지출 TOP 3     │  │  생활비 & 공과금  [탭: 건물명 선택] [+ 항목추가] │
│  최근 3개월 현황     │  │  카드결재                           [+ 항목추가] │
└─────────────────────┘  └──────────────────────────────────────────────────┘
```

### 왼쪽 패널

| 섹션 | 내용 |
|------|------|
| 월 선택 | YYYY-MM 드롭다운, 기본값 당월 |
| 수입 대비 지출 현황 | 수입 / 지출 / 잔액(색상 구분) |
| 주요 지출 TOP 3 | 당월 금액 기준 상위 3개 + 전월 대비 변동(↑/↓/±0) |
| 최근 3개월 현황 | 월·수입·지출 3행 테이블 |

### 오른쪽 패널

| 섹션 | 내용 |
|------|------|
| 고정지출 | 항목명 hover→툴팁, 날짜, 결제수단, 금액, 메모, 편집 |
| 고정이체 & 금융 | 항목명(계좌번호 툴팁), 날짜, 금액 |
| 생활비 & 공과금 | sub_category 탭(건물별), 항목 목록 |
| 카드결재 | 카드사, 결제금액, 전월대비, 정산기간, 결제일 |

---

## 주요 UX

| 동작 | 설명 |
|------|------|
| 금액 수정 | 행 클릭 → 인라인 input 전환 → Enter/blur 저장 |
| 항목 추가 | [+ 항목추가] 버튼 → 모달 |
| 항목 비활성화 | 삭제 대신 is_active=false |
| 월 초기화 | 신규 월 첫 접근 시 "이전 달 복사" 버튼 → default_amount 기준 일괄 생성 |
| 잔액 색상 | 잔액 ≥ 0 → 파란색, 잔액 < 0 → 빨간색 |
| 툴팁 | 모든 항목 행 hover 시 상세 정보 표시 |
